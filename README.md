# Computer-vision-course-design  
计算机视觉课程设计（手掌检测）  
## 实验环境  
IDE：Visual Stdio 2019  
opencv：4.1  
## 实验内容  
+ 测量各个手指的长度与宽度  
+ 测量虎口的角度  
+ 测量手掌宽度  
+ 测量手腕宽度
